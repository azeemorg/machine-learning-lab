# Machine Learning Lab

This repository contains Machine Learning laboratory experiments implemented using Python, Scikit-Learn, and Jupyter Notebook. The experiments cover fundamental Machine Learning algorithms, model training, evaluation techniques, and data visualization.

## Experiments

### Experiment 1: Linear Regression

* Simple Linear Regression
* Data Preprocessing
* Model Training and Prediction
* Performance Evaluation

### Experiment 2: Logistic Regression

* Binary Classification
* Data Preprocessing
* Model Training and Testing
* Accuracy Evaluation

### Experiment 3: Decision Tree and Random Forest

* Decision Tree Classifier
* Entropy and Gini Index
* Information Gain
* Random Forest Classifier
* Confusion Matrix
* Model Accuracy Evaluation

### Experiment 4: Naive Bayes

* Gaussian Naive Bayes (Iris Dataset)
* Multinomial Naive Bayes (IMDb Sentiment Analysis)
* TF-IDF Vectorization
* Confusion Matrix Visualization
* Classification Report
* Accuracy Evaluation

### Experiment 5: K-Nearest Neighbors (KNN)

* KNN Classification
* Euclidean Distance Calculation
* Train-Test Split
* Confusion Matrix
* Classification Report
* Prediction on New Data
* Model Evaluation

## Installation

Clone the repository:

```bash
git clone https://github.com/azeemorg/machine-learning-lab.git
```

Navigate to the project directory:

```bash
cd machine-learning-lab
```

Create and activate a virtual environment:

```bash
python -m venv venv
```

Windows:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```
